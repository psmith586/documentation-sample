Introduction

Python is an easy to learn, powerful programming language. It has efficient high-level data structures and a simple but effective approach to object-oriented programming

The Python interpreter and the extensive standard library are freely available in source or binary form for all major platforms from the Python Web site, https://www.python.org/, and may be freely distributed.

  • The Python interpreter is easily extended with new functions and data types implemented in C or C++ (or other languages callable from C)
  • Python is also suitable as an extension language for customizable applications.

    Hello World
    To get started with writing Python, open the Notepad and write your first "Hello world" Python code: def helloWorld(fName): print('Hello World, ' + fName) fName = 'John' helloWorld(fName) Output: Hello World, John Select the code and run it in an interpreter.     Variables

    In Python, variables are a storage placeholder for texts and numbers.

    Python is dynamically typed, which means that you don't have to declare what type each variable is. The variable is always assigned with the equal sign, followed by the value of the variable.

    Some examples of legal names are fName, width, and sum99. A variable cannot begin with an integer and must begin with a lowercase letter.

    Declaring variables
    You can use a variable in several ways:

    For example, x = 10. This syntax can be used to declare both local and global variables.

    You can declare a variable as an equation as well: x = 5 * 2

    You do not need to declare variable before using it within your code and it can also be changed later in the program if necessary. x = 10 print(x) print() x = x + 10 print(x) Output: 10 20 Each variable in Python is considered an object.

    Variable Scope

    A variable that is declared within a function is known as a local variable because it can only be used inside that function. A local function serves as a container for values that are to be manipulated within the function.

    A variable declared out side all functions within your code is known as a global variable and can be used by the rest of the code. This is helpful when using an object within several different functions.

    x = 10 def main(): def foo(bar): ... main()

    An example of a global variable outside our main function which can be used by all other functions.

    def main(): def foo(bar): x = 10 ... def bar(foo): ...

    In this example, x is a local variable within foo(). Calling bar(x) would return an error because that function cannot access the variable 'x'.

    Strings

    Python can also manipulate strings, which can be expressed in several ways. They can be enclosed in single quotes ('...') or double quotes ("...") with the same result.

    Strings can be concatenated (glued together) with the + operator

    x = 'foo' y = 'bar' print (x + y) Output: foobar
    Lists

    Python knows a number of compound data types, used to group together other values.

    The most versatile is the list, which can be written as a list of comma-separated values (items) between square brackets.

    myList = [1, 2, 3, 4] print(myList) Output: [1, 2, 3, 4]

    Lists might contain items of different types, but usually the items all have the same type.

    Like strings (and all other built-in sequence type), lists can be indexed and sliced For example:

    myList = [1, 2, 3, 4] print(mylist[1]) Output: [2] print(myList[-3:]) Output: [2, 3, 4]

    • if...else statement
    Use an if/else statement to execute a block of code if a logical clause is true. use the elif keyword to present more logical clauses and end with the else keyword if the logical clauses are false. An if statement looks like this: if x == 10: print('x = 10') elif x > 10: print('x > 10') else: print('x < 10') There can be zero or more elif parts, and the else part is optional. The keyword 'elif' is short for 'else if', and is useful to avoid excessive indentation. An if...elif...elif...sequence is a substitute for the switch or case statements found in other languages.
    while statement
    A while loop continues to execute its code block until a condtion is met. It looks like this: x = 8 while x < 10: x += 1 print(x) Output: 9 10

    The condition test occurs before statement in the loop is executed. If the condition returns true (meaning x is still less than 10), statement is executed and the condition is tested again. If the condition returns false (when x is equal to or greater than 10), execution stops and control is passed to the statement following while.

    With each iteration, the function increments x. Therefore, x takes on the following values:

  • After the first pass: x = 9
  • After the second pass: x = 10

    • After completing the second pass, the condition 'x < 10' is no longer true, so the loop terminates.
